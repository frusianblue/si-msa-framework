# 적용 — 인증·아이덴티티 구성 가이드(오너용 결정/레시피)

레포 루트에서:
```bash
unzip -o auth-composition-guide.zip
git status   # docs/guide/AUTH_COMPOSITION_GUIDE.md (신규), docs/00_INDEX.md (링크 추가)
```

## 담은 것
- AUTH_COMPOSITION_GUIDE.md : 결정 카테고리(인증 방식/상태 관리/소셜/MFA/엔터프라이즈)별 옵션 +
  구현 상태(✅/🟡/⬜) + 표준 추천 조합 6종(R1~R6) + 미구현 추가 절차.
- 00_INDEX.md : 오너 섹션·주제 색인에 위 가이드 링크 추가.

## 전제
1~4단계 문서 구조 기준. 링크 전부 해소(검증 완료).
